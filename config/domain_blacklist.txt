# How matching works:
# - Plain domains match that hostname and all subdomains.
#   Example: google.com

#   Matches any URL containing that exact fragment.

# - Block an entire site:

#
# - Block only one section of a site:
#   google.com/about/careers
#
# - Re-enable a blocked site:
#   Delete the line or comment it out with "#"
